## 用户管理中心

#### by 羽雲: https://github.com/yunhaoyu

### 项目简介:
一个企业中最最最常用的系统一一用户中心，其实就是一个简单的“用户管理系统”，实现了用户注册、登录、查询等基础功能。

### 技术选型
#### 前端
主要运用阿里Ant Design 生态:
- HTML + CSS + JavaScript 三件套
- React开发框架
- Ant Design Pro项目模板
- Ant Design 端组件库
- Umi 开发框架
- Umi Request 请求库
#### 后端
- Java 编程语言
- Spring + SpringMVC + SpringBoot 框架
- MyBatis + MyBatis Plus 数据访问框架
- MySQL 数据库
- jUnit 单元测试库

### 编译环境:
- JDK 1.8
- MySQL 8.0.30
